# MySQL

```sql
SHOW DATABASES; -> 현재 있는 데이터베이스 조회
USE database_name; -> 사용할 데이터베이스 지정

SHOW TABLES; -> 데이터베이스 내 테이블 조회
DESC table_name; -> Describe table
```
